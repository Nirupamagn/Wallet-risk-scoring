Wallet Risk Scoring Project

Overview:
The goal of the Wallet Risk Scoring project is to evaluate Compound protocol wallet addresses and provide a risk score based on their transaction history.  This project helps users understand their exposure in the DeFi sector by evaluating each wallet's risk profile using on-chain data.


Features:
Obtain transaction information from the Compound protocol.
Clean up and process the transaction data.
Determine risk scores using a variety of features.
